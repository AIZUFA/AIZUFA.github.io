BoraFlow AI
End-to-end deep learning pipeline for human motion recognition using BTS choreography as a stress test. Achieved 87.90% real-time confidence with Stacked LSTMs trained on 85,000+ frames. Published as a research paper with healthcare applications.
TensorFlow YOLOv8 MediaPipe OpenCV Research Paper
02
Affective Fragrance Synthesis
Neurosymbolic AI framework combining ocular biometrics, facial affect recognition (92% accuracy), and LLM orchestration to deliver real-time personalized aromatic interventions. Submitted to FIT 2026.
LLM Integration OFKNet ESP32 Raspberry Pi FIT 2026

BU Intercampus Secure Network
Designed and simulated a hierarchical multi-campus network connecting all 4 Bahria University campuses with VLAN segmentation, OSPF dynamic routing, and Extended ACLs. Verified in Cisco Packet Tracer.
Cisco Packet Tracer OSPF VLAN ACL

SRAP — Swarm Robotics Protocol
Designed a custom lightweight application layer protocol (SRAP) for swarm robotic systems using Python UDP sockets. Achieved zero packet loss with 3-second heartbeat and real-time Euclidean distance tracking.
Python UDP Sockets JSON Threading
